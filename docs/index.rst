=============
django-modals
=============

alert/confirm/prompt modal dialogs you can customize via django template

Installation
------------

Install with pip::

    pip install django-modals

Then declare the app in your settings.py ::

    INSTALLED_APPS = [
    ...
        'django_modals',
    ]
